Project Overview

Developed a robust web scraper to automatically extract Standard Industrial Classification (SIC) codes for all public companies directly from the SEC website. This tool streamlines the process of gathering industry classification data for financial analysis and research.

Key Features

  • Automated extraction of SIC codes from SEC EDGAR database
  • Handling of rate limiting and pagination
  • Data validation and error handling
  • Export functionality to various formats

Technologies Used

  • Python
  • BeautifulSoup4
  • Requests
  • Pandas
  • Web Scraping Best Practices
